# User Stories (เรื่องราวผู้ใช้งาน)
**Project:** Real-time Collaborative To-Do List


---

1. "ในฐานะ **ผู้ใช้งานใหม่**, ฉันต้องการ **กรอกชื่อเล่น (Display Name) ก่อนเข้าสู่ระบบ**, เพื่อให้ **ระบบบันทึกชื่อของฉันไว้ใช้แสดงความเป็นเจ้าของงานต่างๆ**"

2. "ในฐานะ **สมาชิกทีม**, ฉันต้องการ **เพิ่มรายการงานใหม่โดยพิมพ์ข้อความแล้วกด Enter**, เพื่อให้ **งานถูกส่งเข้าไปในระบบและแสดงผลในรายการทันที**"

3. "ในฐานะ **ผู้ใช้งาน**, ฉันต้องการ **เห็นรายการงานของเพื่อนเด้งขึ้นมาทันทีโดยไม่ต้องรีเฟรชหน้าจอ**, เพื่อให้ **ฉันได้รับข้อมูลที่เป็นปัจจุบันที่สุด (Real-time) ในการทำงานร่วมกัน**"

4. "ในฐานะ **ผู้รับผิดชอบงาน**, ฉันต้องการ **เปลี่ยนสถานะงานเป็น 'กำลังทำ (In Progress)' ผ่านเมนู Dropdown**, เพื่อให้ **แถบสีของงานเปลี่ยนเป็นสีส้มและทีมรู้ว่างานนี้กำลังถูกดำเนินการ**"

5. "ในฐานะ **ผู้ตรวจสอบงาน**, ฉันต้องการ **เปลี่ยนสถานะงานเป็น 'เสร็จสิ้น (Done)'**, เพื่อให้ **ชื่องานถูกขีดฆ่าและแสดงเป็นสีเขียวเพื่อยืนยันความสำเร็จ**"

6. "ในฐานะ **ผู้ใช้งาน**, ฉันต้องการ **กดปุ่ม [Edit] เพื่อแก้ไขชื่องานแบบ Inline (พิมพ์แก้ในบรรทัดนั้นเลย)**, เพื่อให้ **ฉันสามารถแก้ไขคำผิดได้สะดวกรวดเร็วโดยไม่ต้องไปหน้าอื่น**"

7. "ในฐานะ **ผู้ใช้งาน**, ฉันต้องการ **กดปุ่ม [Cancel] เพื่อยกเลิกการแก้ไข**, เพื่อให้ **ข้อความกลับไปเป็นเหมือนเดิมหากฉันเปลี่ยนใจหรือไม่ต้องการแก้ไขแล้ว**"

8. "ในฐานะ **ผู้ดูแลบอร์ด**, ฉันต้องการ **กดปุ่ม [Del] เพื่อลบงานที่ยกเลิกแล้ว**, เพื่อให้ **กระดานงานมีความสะอาดตาและจัดการง่าย**"

9. "ในฐานะ **ผู้ใช้งาน**, ฉันต้องการ **ให้ระบบถามยืนยัน (Confirm Alert) ก่อนที่จะลบงานจริง**, เพื่อให้ **ป้องกันความผิดพลาดจากการเผลอกดปุ่มลบโดยไม่ตั้งใจ**"

10. "ในฐานะ **สมาชิกทีม**, ฉันต้องการ **เห็นชื่อผู้สร้าง (Created By) และเวลาที่สร้างบนการ์ดงาน**, เพื่อให้ **ฉันรู้ที่มาที่ไปของงานและรู้ว่าใครเป็นคนมอบหมาย**"
